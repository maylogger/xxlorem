XXLOREM.js
一用就憤怒的中文假字生產器

這是什麼？

XXLOREM.js 是一款針對 前端設計師 打造的 中文假字生產器。

假字是什麼？能吃嗎？

假字是讓設計師用來填充版面的內容物，作為提案和設計過程中的暫時內容，而非讓大眾閱讀。

語料的來源是？

XXLOREM.js 的語料使用 靠北設計師 的匿名靠北內容。我們從中篩選了一些內容，並以程式隨機打亂後重新組合，作為假字使用。

有開源嗎？

你可以到 Github fork 這支 Plugin，歡迎任意複製、更改。

    使用方法

    XXLOREM.js 是以 jQuery 為基礎編寫，使用時需先引用 jQuery
    請在你的網頁加上以下語法：

    <script src="//code.jquery.com/jquery-1.11.3.min.js"></script>
    <script src="//xxlorem.unfoldgroup.com/plugin/xxlorem-1.1.js"></script>

    XXLOREM.js 提供了兩種方法，讓你可以快速將假字套用到頁面中：

    方法一：以 jQuery 套用假字於特定物件上
    Javascript
    $('.example').xxlorem();
    $('.example').xxlorem(3);
    $('.example').xxlorem('10w');
    $('.example').xxlorem('3-5','p');
    方法二：直接在想要填入假字的物件加上 class="xx"
    Inline html
    <p class="xx"></p>
    <p class="xx(3)"></p>
    <p class="xx(10w)"></p>
    <p class="xx(3-5,p)"></p>

    注意：當你在使用像 Slim 這類的樣板語言編寫頁面時，可能無法以圓括號的方式添加參數，但是好家在，XXLOREM.js 兼容以「- (減號)」或「_ (底線)」作為參數的串接符號：

    <p class="xx-5"></p>
    <p class="xx_3_p"></p>
    <p class="xx-name"></p>

    參數設置

    前述的兩種使用方法，都可以藉由設置不同的參數，來產生不同類型或長度的假字。

    採用 方法一 時，您可以用 $(elment).xxlorem(params) 方式來設置參數；在 方法二 中，則可以用增加括弧和數值的方式 <p class="xx(params)"></p> 來設置參數。

    XXLOREM.js 提供了多種類型的假字如下表：

    類型 參數名 簡寫 說明
    文字 word , words w 產生一個或多個文字
    句子 sentence , sentences s 產生一個或多個句子
    段落 paragraph , paragraphs p 產生一個或多個段落文字
    姓名 name n 產生一個假姓名
    姓 last name ln 產生一個假姓氏
    名 first name fn 產生一個假名字
    姓名(拼音) name pinyin np 產生一個拼音假姓名
    姓(拼音) last name pinyin lnp 產生一個拼音假姓氏
    名(拼音) first name pinyin fnp 產生一個拼音假名字
    Email email , mail e , m 產生一個假 email

    通常來說，只需用一個參數就可以完成設置假字的類型或長度 (or range)；或者，你也可以用縮寫的方式，同時設置假字類型和長度（例如 xx(3p) 會產生 3 個段落）：

    Javascript
    $(elment).xxlorem('name');
    $(elment).xxlorem(5);
    $(elment).xxlorem('10w');

    Inline html
    <span class="xx(email)"></span>
    <span class="xx(lnp)"></span>
    <span class="xx(3p)"></span>

    注意：直接寫在 HTML 標籤中，參數前後不需 ' 或 " 引號！

    當你要需要產生多個文字、句子或段落時，則可以使用 2 個參數來設置假字類型和長度 (or range)，請以逗號隔開參數，參數順序不影響結果：

    Javascript
    $(elment).xxlorem('sentences', 10);
    $(elment).xxlorem('3-5', 'p');

    Inline html
    <span class="xx(words,10-20)"></span>
    <p class="xx(3,p)"></p>

    注意：直接寫在 HTML 標籤時，逗號前後請勿留空白符號！

    在產生段落 (paragraphs) 時，可彈性的使用第三個參數。

    第三個參數為設置換行符號，預設的換行符號為 \n\n，可依照需求替換，例如：

    Javascript
    $(elment).xxlorem('3-5', 'paragraphs', '<br><br>');

    Inline html
    <p class="xx(3-5,p,<hr>)"></p>

    除非要產生隨機長度的假字，否則我們建議以縮寫、僅使用兩個參數的方式設置段落：

    Javascript
    $(elment).xxlorem('3p', '<br><br>');

    Inline html
    <p class="xx(6p,<hr>)"></p>

    使用範例

    給我 1 個中文字：

    Preview

    Code <p class="xx(w)"></p>

    你也可以這樣寫：xx(word)、xx(1 word)

    馬上給我 19 個字：

    Preview

    Code <p class="xx(19 words)"></p>

    你也可以這樣寫：xx(19w)、xx(words,19)、xx(19_word)、xx(19-words)

    標題大概 10 到 20 個字左右：

    Preview

    Code <p class="xx(10-20,w)"></p>

    你也可以這樣寫：xx(10~20,words)、xx(w,10 20)

    這邊太空了，麻煩隨便來個句子：

    Preview

    Code <p class="xx"></p>

    你也可以這樣寫：xx(sentence)、xx(1)、xx(s)

    太短太短！我需要大概 3 句話的文字：

    Preview

    Code <p class="xx(3s)"></p>

    你也可以這樣寫：xx(3 sentences)

    長度大概 5 到 10 句：

    Preview

    Code <p class="xx(5-10)"></p>

    你也可以這樣寫：xx(5~10)、xx(5,10)

    這邊放一個段落：

    Preview

    Code <p class="xx(p)"></p>

    你也可以這樣寫：xx(paragraph)

    我跟你說過幾次這邊要 3 個段落，每個段落之間要有空白：

    Preview

    Code <p class="xx(3p,<br><br>)"></p>

    你也可以這樣寫：xx(3 paragraphs,<br><br>)

    我要 3 到 5 個段落：

    Preview

    Code <p class="xx(3-5,p,<br><br>)"></p>

    你也可以這樣寫：xx(p,3~5,<br><br>)

    來個人名：

    Preview

    Code <p class="xx(name)"></p>

    你也可以這樣寫：xx(n)

    來個拼音的假姓名：

    Preview

    Code <p class="xx(name pinyin)"></p>

    你也可以這樣寫：xx(np)、xx(name-pinyin)

    來個 Email

    Preview

    Code <p class="xx(email)"></p>

    你也可以這樣寫：xx(e)、xx(mail)、xx(m)
